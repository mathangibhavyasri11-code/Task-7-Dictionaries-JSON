# Task 7 - Dictionaries & JSON Handling

## Objective
This project demonstrates how to:
- Create and manage dictionaries in Python
- Perform CRUD operations
- Convert dictionary to JSON
- Save JSON to a file
- Read JSON data back into Python

## Tools Used
- VS Code

## Files Included
- student_records.py (Main Python Script)
- students.json (Generated JSON file)
